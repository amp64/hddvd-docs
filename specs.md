# HD DVD Specifications
This is the only known archive of these specifications. They were generated by scanning paper copies into PDFs, OCR-ing them, 
then fixing up the page numbers.

## Location
The PDFs are stored in this repository in the [Releases](/releases) area. 
(If you know of a better way of integrating PDFs into a GitHub repository, please let me know).
