# Jumpstart

Jumpstart is a set of tools released by Microsoft to help developers create HD DVD content. The tools were released in 2005, and were available for download from Microsoft. The tools were later removed from the website, but have been recovered from the Internet Archive.

For convenience I have pre-expanded the .CHM and .PNG files which are most often useful.

These files are all Copyright Microsoft Corporation, and are provided here for archival purposes.

- [Jumpstart Installer](HDDVDJumpstart.msi) Digitally signed by Microsoft Corporation, will likely complain about not finding Media Player (because it's not 2008 any more).
- [HD DVD Script Object Model](HD_DVD_Script_Object_Model.png) A diagram of the script object model.
- [HD DVD Programming Guide](HDDVDIJ.CHM) The main documentation for the Jumpstart tools. Uses the Html Help Engine.
